<h2>git/git-cppcheck<h2>
This script prints out cppcheck results of code changes.<br>
The code changes could be uncommitted (by default) or commit specified.<br>
It returns 1 if there're error or warning reports.<br>
<br>
Usage: git-cppcheck [\<commit|file\>...]<br>
<br>
Examples:<br>
git-cppcheck        -- check uncommitted code changes<br>
git-cppcheck HEAD   -- check code changes in latest commit<br>
git-cppcheck 03c58d -- check code changes in commit<br>
git-cppcheck *.cc   -- check files<br>
